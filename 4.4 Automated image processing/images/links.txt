Digitize > used in Multi-otsu to make a new picture using the thresholds
	its like making a new array with zeros and filling it.
	regions = np.digitize(img, bins = thresholds)

flat > to flatten an array from 2D to 1D to plot a histogram. eg. img.flat
	this is a Generator like range

ravel() > to flatten an array from 2D to 1D, but its not a generator
	eg. img.ravel()
